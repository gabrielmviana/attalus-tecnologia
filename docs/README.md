# Documentação do Projeto ATTALUS

Este diretório contém toda a documentação oficial referente ao Projeto ATTALUS, incluindo infraestrutura, redes, arquitetura, padrões internos e diretrizes operacionais.

## Estrutura

- **network/** — Documentação relacionada à rede e seus serviços.
- **infra/** — Guias técnicos e informações sobre servidores, virtualização e sistemas.
- **security/** — Políticas e diretrizes de segurança, controles e hardening.
- **operations/** — Procedimentos de operação, suporte e resposta a incidentes.

A documentação será expandida conforme o projeto evolui.
