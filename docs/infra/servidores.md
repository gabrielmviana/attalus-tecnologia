# Servidores - Projeto ATTALUS

Este documento descreve a infraestrutura de servidores do ATTALUS, incluindo hardware, topologia e função de cada máquina.

## Infraestrutura Inicial
- 1x Servidor Dell (em negociação)
- 1x Notebook Dell (host de testes)
- VirtualBox como hypervisor inicial

## Funções Previstas
- Servidor de arquivos
- Servidor de monitoramento
- Servidor DNS interno
- Servidor de autenticação (futuro)
