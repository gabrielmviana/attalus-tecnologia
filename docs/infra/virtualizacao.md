# Virtualização - Projeto ATTALUS

A virtualização será o núcleo do laboratório ATTALUS, permitindo a criação e gerenciamento de múltiplos serviços.

## Ambiente Inicial
- VirtualBox (host local)
- Planejamento futuro para Proxmox

## Objetivos
- Criar ambiente modular e escalável.
- Facilitar snapshots e rollback.
- Criar topologias para estudo de redes e infraestrutura.

## Próximos passos
- Documentar VMs iniciais.
- Definir templates base (Linux/Windows).
- Planejar migração para Proxmox.
