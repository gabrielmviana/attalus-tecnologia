# VLANs - Projeto ATTALUS

Este documento descreve o planejamento e organização das VLANs do ambiente ATTALUS.

## Objetivos
- Segmentar a rede para maior segurança.
- Garantir isolamento entre serviços.
- Organizar o tráfego interno.

## Estrutura Inicial Proposta
- VLAN 10 — Administração
- VLAN 20 — Servidores
- VLAN 30 — Usuários
- VLAN 40 — LAB / Homelab
- VLAN 50 — IoT / Rede isolada
