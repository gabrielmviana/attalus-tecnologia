# Mikrotik - Projeto ATTALUS

O Mikrotik é o core da infraestrutura de rede do ATTALUS, sendo responsável por:

- Roteamento
- Firewall
- VLANs
- VPNs
- DHCP/DNS interno

## Configurações Iniciais
- Criação de VLANs
- Configuração do firewall básico
- DHCP por VLAN
- Ajuste de bridge e interfaces
