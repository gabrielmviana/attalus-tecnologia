# DHCP - Projeto ATTALUS

O serviço DHCP é responsável por entregar configuração automática de rede aos dispositivos conectados no ambiente.

## Definições Iniciais
- Faixas de IP por VLAN
- DNS padrão
- Gateway padrão
- Reservas estáticas (para servidores e equipamentos críticos)

## Equipamentos Envolvidos
- Mikrotik RouterOS (principal servidor DHCP)
