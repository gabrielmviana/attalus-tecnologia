# DNS - Projeto ATTALUS

O DNS será responsável pela resolução de nomes internos da rede ATTALUS.

## Objetivos
- Facilitar o acesso a serviços internos.
- Organizar o namespace corporativo.
- Criar zona DNS local: `attalus.local`

## Entradas Iniciais

- `gw.attalus.local`
- `nas.attalus.local`
